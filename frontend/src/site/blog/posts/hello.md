For a long time I only ever wrote in code — commits, docs, the occasional long Slack message.
This is me trying something different: writing for myself, out loud.

I don't want this to be a "tech blog." Some of it will be about the things I build and the messy
engineering behind them, sure. But just as often I want to write about the rest of it — things I'm
reading, places I've been, small wins and dumb mistakes, opinions I'm still forming, and whatever
happens to be on my mind that week.

No fixed theme. No schedule. No pretending I have it all figured out.

A few things I already know I want to write about:

- **Life stuff** — habits, decisions, things I'm learning outside of work.
- **Experiences** — trips, people, moments worth remembering.
- **Hobbies & interests** — the things I do just because I enjoy them.
- **And yes, some tech** — when I build something I'm proud of, I'll share how.

If you're here, thanks for reading. Say hi if something resonates.
